Sphinx Installer CHANGELOG
==========================


v0.0.2     28-Jul-2021
======================
Updated Sphinx to v4.1.2.
Updated `sphinx-rtd-theme` to v0.5.2.
Added additional Sphinx extensions:
- sphinx-press-theme
- sphinx-last-updated-by-git

Author: Niko Križnik


v0.0.1     27-Mar-2021
======================
Added Inno Setup based installer that downloads and installs
Python and pip. The script also installs the required pip
packages that support Sphinx.
The Sphinx binaries can be accessed via the `SPHINX_INSTALL_PATH`
environment variable.

Author: Niko Križnik
